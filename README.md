\# <normal\_force\_c\_reinhardtii\_c\_sorokiniana\_femtotool>



This repository contains the Python scripts and raw data for:

<Single-cell strength of nitrogen limited Chlamydomonas reinhardtii and Chlorella soro-kiniana>



\## Contents

* `scripts/` - python Analysis scripts
* `data/`raw data files



\## Important information



The path to the raw data Needs to be adjusted for your path in the scripts
